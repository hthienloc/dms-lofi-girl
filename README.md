# Lofi Radio for DankMaterialShell

A beautiful online Lofi radio streaming plugin for [DankMaterialShell](https://github.com/DankMaterialShell). 

## Features

- **Curated Stations**: Listen to Lofi Girl, Chillhop Music, and more with one click.
- **Animated Pill**: Features a radio icon and dancing audio bars when music is playing.
- **Volume Controls**: Smooth volume adjustment via slider or mouse wheel on the bar.
- **Fast Buffering**: Uses `mpv` for high-performance audio streaming.
- **Deep Integration**: Matches the DMS theme perfectly.

## Installation

1. Clone this repository to your DMS plugins directory:
   ```bash
   cd ~/.config/DankMaterialShell/plugins
   git clone https://github.com/yourusername/dms-lofi-radio lofiRadio
   ```
2. **Requirements**: Ensure you have `mpv` installed on your system:
   ```bash
   sudo pacman -S mpv  # Arch
   sudo apt install mpv # Ubuntu/Debian
   ```

## Usage

- **Click Icon**: Opens the radio selection grid.
- **Right-Click Icon**: Quickly Mute/Unmute.
- **Mouse Wheel on Icon**: Adjust volume up/down.
- **Select Station**: Click any station tile in the popout to start/stop streaming.
